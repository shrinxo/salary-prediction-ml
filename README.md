# Salary Prediction using Machine Learning

This project predicts salary based on years of experience using Linear Regression.

Dataset
- YearsExperience
- Salary

Steps
- EDA
- Train-test split
- Linear regression
- Evaluation (MAE, R²)

Results
- MAE: ~6.2k
- R²: 0.90

Tech Stack
- Python
- Pandas
- Scikit-learn
